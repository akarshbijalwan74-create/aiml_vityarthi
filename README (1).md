# SmartTicket-AI

## Overview
An intelligent command-line tool that classifies customer support tickets into categories (Technical, Billing, Account) using Natural Language Processing.

## Features
- **ML-Powered:** Uses Scikit-Learn for instant classification.
- **Secure:** Admin authentication required.
- **Auditable:** Logs all activities to a local file.

## Technologies
- Python 3.8+
- Scikit-learn
- Pandas

## Installation & Running
1. Install dependencies: `pip install -r requirements.txt`
2. Run the app: `python main.py`
3. Log in with `admin` / `admin123`


## Instructions for Testing:
- Type a query like "internet is broken" to see the classification.
