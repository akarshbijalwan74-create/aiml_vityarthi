# Project Statement

## Problem Statement
Customer support teams are overwhelmed by unorganized queries. Manual sorting is slow and error-prone.

## Scope
This project automates the initial categorization of text-based support tickets using Machine Learning.

## Target Users
IT Support Administrators and Helpdesk Managers.

## High-Level Features
1. Secure User Authentication.
2. Real-time text classification using Naive Bayes.
3. Automated activity logging for audit trails.
